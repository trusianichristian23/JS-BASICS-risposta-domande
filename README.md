# JS-BASICS-risposta-domande

1. Cos'è un algoritmo?
   Un esempio di algoritmo nella vita quotidiana è prepararsi per andare a scuola: Svegliarsi, Alzarsi dal letto, Lavarsi, Vestirsi, Fare colazione, Prendere lo zaino, Uscire di casa
   Questa sequenza è un algoritmo perché: i passi sono ordinati, ogni azione è chiara; se segui sempre gli stessi passi ottieni sempre lo stesso risultato, a un certo punto finisce.
2. Cos'è una variabile?
   E' un contenitore che può essere utilizzato più volte e può contenere qualsiasi cosa.
   
3. Undefined e null sono la stessa cosa?
   Undefined è mancanza di valore, null è un valore che viene assegnato intenzionalmente.
